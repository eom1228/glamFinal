# glamFinal

### 구현한 기능
  - 더미데이터를 이용해 상태 업데이트하고 컨텍스트API를 활용한 상태관리 
  - 메인페이지:
    - 좋아요, 싫어요, 관심 버튼 이벤트 핸들러 등록 및 상태관리 
    - 슬라이더안에 프로필 사진 등록 
    - 실시간 프로필에 별점 주기 기능 구현 
  - 마이페이지: 
    - sass로 UI 구현 
  - 푸터를 통해 메인페이지와 마이페이지 간 페이지 전환 구현
